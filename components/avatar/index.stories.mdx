# Avatar

The 'Avatar' component is used to display a user's profile picture. If no image
is provided then a users initials will be displayed.

If the image is too large then it will be scaled down to fit within the
component's bounds.

If no user is passed to the component then it will display a default image.

export const Template = (args) => (
  <div style={{ height: '200px' }}>
    <Avatar {...args} />
  </div>
);
