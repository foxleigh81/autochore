# Input

The input component is used to collect user input it is used in conjunction with
the [inputContainer](?path=/docs/atom-inputcontainer) component and (depdending on circumstances), the [form]()
component.

### Mobile only types

The following types do not do anything exciting on desktop devices,
however on a mobile device, they will change the users keyboard to
include extra features. Such as a number pad or a '.com' button.

Appearance-wise, they all appear as a reuglar text input, so it makes sense to
group them together, view this page on mobile to see the differences.
