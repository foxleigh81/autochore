# LoginForm

A component that renders a login form. It can be used to login with email or federated
providers. The component also supports pre-populating the email address and password
fields.


<Canvas withSource="open">
  <Story name="Default">
    <LoginForm loginWithEmail={() => {
        alert("Login with Email")
      }} />
  </Story>
</Canvas>

<Canvas withSource="open">
  <Story name="Error">
    <LoginForm passwordError={"Incorrect password"} loginWithEmail={() => {
        alert("Login with Email")
      }}/>
  </Story>
</Canvas>

### Pre-population

You can pre-populate the email address and password fields by passing them as props.

<Canvas withSource="open">
  <Story name="Pre-populated Email">
    <LoginForm email={"joe@bloggs.co.uk"} loginWithEmail={() => {
        alert("Login with Email")
      }}/>
  </Story>
</Canvas>

### Fedarated Login

You can allow users to use a federated login provider by passing an array of
'federatedProvider' objects as a prop. At the moment, only Google is supported.

<Canvas withSource="open">
  <Story name="Login with Google">
    <LoginForm
      email={"joe@bloggs.co.uk"}
      loginWithEmail={() => {
        alert("Login with Email")
      }}
      federatedProvider={[{
        name: "Google",
        logo: <FcGoogle />,
        loginWithProvider: () => {
          alert("Login with Google")
        }
      }]}
      />
  </Story>
</Canvas>
